mindquantum.core.circuit.Circuit.compress()

        删除所有未使用的量子位，并将量子位映射到`范围（n_qubits）`。

        样例:
            >>> from mindquantum import qft
            >>> qft([0, 2, 4])
            q0: ──H────PS(π/2)────PS(π/4)─────────────────────────@──
                          │          │                            │
            q2: ──────────●──────────┼───────H────PS(π/2)─────────┼──
                                     │               │            │
            q4: ─────────────────────●───────────────●───────H────@──
            >>> qft([0, 2, 4]).compress()
            q0: ──H────PS(π/2)────PS(π/4)─────────────────────────@──
                          │          │                            │
            q1: ──────────●──────────┼───────H────PS(π/2)─────────┼──
                                     │               │            │
            q2: ─────────────────────●───────────────●───────H────@──

mindquantum.core.circuit.Circuit.rx(para, obj_qubits, ctrl_qubits=None)

        添加接收门。

        参数:
            para (Union[dict, ParameterResolver]): `RX`门的参数。
            obj_qubits (Union[int, list[int]]): `RX`门的对象量子位。
            ctrl_qubits (Union[int, list[int]]): `RX`门的控制量子位。默认值：None。

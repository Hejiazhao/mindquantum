mindquantum.core.parameterresolver.ParameterResolver.params_name

        获取参数名称。

        返回：
            list，参数名称的列表。

        样例：
            >>> from mindquantum import ParameterResolver
            >>> pr = ParameterResolver({'a': 1, 'b': 2})
            >>> pr.params_name
            ['a', 'b']b']
